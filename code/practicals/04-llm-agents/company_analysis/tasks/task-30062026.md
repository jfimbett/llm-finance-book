
Create the following agents and skills in order to write a report about a publicly traded company

Agents

Name: Financial Analyst
Role: An expert graduate financial analyst specialized in publicly traded companies. 

Name: Generalist
Role: An expert macroeconomists that understands what are they key financial variables that a financial analyst should focus on when deciding to write a company report. 

Name: Data Retriever
Role: Given the name of a company, the data retriever retrieves the latest 10-K of the company from the EDGAR filings. Given the name of the company, the agent retrieves the CIK, and the latest 10-K. This agent has access to a tool that retrieves automatically the filings from EDGAR. 
    

Name: Data Scientist
Role: Clean and prepare the financial data retrieved by the Data Retriever agent for other agents to process. 

Name: Report Writer
Role: Given the financial analysis of a company, writes a 2 page professional corporate report to transmit to the portfolio manager. 

Skills

Name: Company Report
Description:
Given a company name, trigger the following agents in order

- Generalist
- Data Retriever
- Data Scientist
- Financial Analyst
- Report Writer

Expected Output: A simple but professional website with the report.
