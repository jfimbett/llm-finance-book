\section{Classical Portfolio Theory and Its Limits}

\begin{frame}{Classical Portfolio Theory and Its Limits}
  \begin{itemize}
    \item Markowitz mean-variance optimization defines the efficient frontier of risk-return trade-offs
    \item The Black-Litterman model extends Markowitz by blending market equilibrium with investor views
    \item Classical models rely on stable covariance estimates that break down in crisis periods
    \item LLMs can enrich inputs but cannot overcome fundamental estimation uncertainty
  \end{itemize}
\end{frame}

\section{LLMs as Alternative Data Processors}

\begin{frame}{LLMs as Alternative Data Processors}
  \begin{itemize}
    \item LLMs can parse earnings calls, analyst reports, and news to extract quantified views
    \item Textual signals can be transformed into return forecasts usable in Black-Litterman
    \item LLM-derived factors augment Fama-French models with narrative-driven dimensions
    \item Data quality and prompt engineering are critical for reliable factor construction
  \end{itemize}
\end{frame}

\section{LLM-Guided Algorithmic Trading}

\begin{frame}{LLM-Guided Algorithmic Trading}
  \begin{itemize}
    \item Sentiment scores derived from LLMs can serve as high-frequency trading signals
    \item Reinforcement learning agents can use LLM outputs to optimize order execution
    \item Market efficiency limits predictability: LLM signals decay rapidly in liquid markets
    \item Combining LLM signals with price-based signals requires careful signal weighting
  \end{itemize}
\end{frame}

\section{Backtesting with LLM Signals}

\begin{frame}{Backtesting with LLM Signals}
  \begin{itemize}
    \item Walk-forward evaluation prevents in-sample overfitting of LLM-based strategies
    \item Transaction costs and market impact must be modeled realistically
    \item Look-ahead bias is a severe risk when LLM outputs embed future information
    \item Survivorship bias inflates backtest returns if the asset universe is not carefully constructed
  \end{itemize}
\end{frame}

\section{Risk Management Applications}

\begin{frame}{Risk Management Applications}
  \begin{itemize}
    \item Textual signals from news and filings can anticipate tail risk events
    \item LLMs enable real-time summarization of portfolio exposures and anomaly detection
    \item Alert systems can monitor regulatory filings and macro news for sudden regime changes
    \item Combining LLM-based risk signals with VaR models improves early-warning capabilities
  \end{itemize}
\end{frame}
