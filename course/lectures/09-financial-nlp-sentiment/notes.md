# Lecture 09: Financial NLP and Sentiment Analysis

## Learning Objectives

After this lecture, students will be able to:
- Identify and describe the main sources of textual financial data and the preprocessing steps required to prepare them for analysis
- Compare lexicon-based, fine-tuned classifier, and LLM-based approaches to financial sentiment analysis
- Analyze sentiment signals derived from diverse text sources including social media, news, earnings calls, SEC filings, and central bank communications
- Construct and evaluate a sentiment-based trading signal using event studies and predictability frameworks
- Design and implement an end-to-end NLP pipeline for financial text at scale
