# Lecture 13: LLM Limitations and Evaluation Rigor in Finance

## Learning Objectives

After this lecture, students will be able to:
- Explain what calibration means and identify overconfidence in LLM financial outputs
- Recognize temporal leakage and look-ahead bias when evaluating LLMs on financial data
- Critically assess stock movement prediction claims made using LLMs
- Apply economic value metrics beyond classification accuracy to evaluate financial models
- Identify hallucination risks in financial LLM applications and describe grounding strategies
