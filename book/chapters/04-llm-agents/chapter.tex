\chapter{LLM Agents and Finance Applications}
\label{ch:llm-agents}

\begin{remark}[Learning Objectives]
After completing this chapter, the reader should be able to:
\begin{enumerate}
  \item Describe the perceive--reason--act cycle and explain how it differs from
        single-turn LLM inference.
  \item Implement a ReAct-style agent loop with tool use and trace its reasoning
        steps on a financial question-answering task.
  \item Design and register custom tools (calculators, database connectors, API
        wrappers) for use in an LLM agent framework.
  \item Distinguish in-context memory, retrieval-augmented memory, and long-term
        persistent memory, and choose the appropriate mechanism for a given task.
  \item Build a retrieval-augmented generation pipeline for long financial documents
        and evaluate it on faithfulness and relevance metrics.
  \item Orchestrate multi-agent workflows using at least one modern framework
        (LangChain, LlamaIndex, or AutoGen) and reason about latency and cost
        trade-offs.
  \item Apply agent-based systems to at least three finance-specific tasks such as
        earnings analysis, automated report generation, and filing search.
  \item Identify adversarial risks (prompt injection, tool misuse) and apply
        human-in-the-loop governance patterns consistent with financial regulation.
\end{enumerate}
\end{remark}
