# Lecture 5: LLMs for Business Valuation

## Learning Objectives

By the end of this lecture, students will be able to:

1. Explain the core business valuation methodologies (DCF, comparables) and identify where LLMs add value in each step.
2. Build an automated pipeline to extract financial statements and tables from SEC EDGAR filings using LLM agents.
3. Compute and forecast free cash flows programmatically, leveraging LLMs for gap-filling and projection.
4. Design tool-augmented agent workflows that reliably perform arithmetic without hallucination.
5. Construct and evaluate scenario analyses (bull/base/bear) and sensitivity tables with LLM assistance.
6. Select comparable companies using both prompt-based and embedding-based approaches.
7. Critically assess the efficiency, accuracy, and failure modes of LLM-assisted valuation pipelines.
