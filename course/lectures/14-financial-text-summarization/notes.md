# Lecture 14: Financial Text Summarization and Information Extraction

## Learning Objectives

After this lecture, students will be able to:
- Describe the information extraction problem in finance and distinguish structured from unstructured text
- Apply named entity recognition techniques to financial documents using pre-trained and fine-tuned transformer models
- Summarize financial documents using both extractive and abstractive approaches
- Extract and validate tabular and numerical data from SEC filings and financial reports
- Evaluate summarization and extraction systems using standard benchmarks and domain-specific metrics
